Ctrl + N : 새로운 파일
Ctrl + S : 저장
Ctrl + D : 같은 단어 찾기 (같이 수정 가능)
Ctrl + C, Ctrl + V : 마우스로 드래그 하지 않고 사용 가능 (복사, 붙여넣기)
Alt + Shift + 방향키(위 아래) : 4와 같은 역할. 훨씬 빠름.
Home, End : 해당 줄 맨 앞으로, 맨 뒤로
Ctrl + Alt + 위아래 : 동시수정
Ctrl + B : 사이드바
Ctrl + / : 주석
Ctrl + [, shift tab : 내어쓰기
Ctrl + ], tab : 들여쓰기
Shift + Del : 한줄삭제
Ctrl + , : settings => word wrap 검색 후 off -> on으로 하면 code가 자동 줄바꿈
